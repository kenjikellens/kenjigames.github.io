CreamInstaller

A sophisticated C# automation tool for managing DLL components across multiple gaming distribution platforms.

Repository Back to Projects

Technical Specification

Built on the .NET framework, CreamInstaller specializes in the automated detection and configuration of DLC-related library files. It bridges the gap between Steam, Epic Games, and Ubisoft libraries, offering a unified interface for system-level tool management.

Platform Agnostic

Intelligent detection of diverse game installation directories across multiple launchers.

Automated DLL Logic

Streamlines library management using recursive directory scanning and precise file handling.
