Technical Engineering

The Great Awakening is built on a custom-engineered vanilla JavaScript Single Page Application (SPA) framework. This technical decision was driven by the need for extreme speed, search engine optimization, and zero external dependencies. By bypassing the bloat of modern libraries, we've created a platform that loads instantaneously and scales effortlessly.

Hash-Based Routing & Fragment Injection

The core of our navigation system is a custom hash-based router. Instead of reloading the entire page, the router intercepts URL changes (e.g., `#home`, `#dossiers`, `#dossier/slug`) and dynamically fetches small HTML "fragments." This process ensures that the user's research flow is never interrupted by full-page refreshes, creating an experience that feels fluid and cohesive.

The Dossier Management System

Our data layer is built on a flat-file JavaScript database, optimized for real-time querying. The Dossier Manager handles the rendering of the Mega Menu and the Categorized Legend automatically, ensuring that new research can be added to the archive with minimal technical friction. This system allows for advanced search capabilities that scan titles, summaries, and keywords simultaneously.
