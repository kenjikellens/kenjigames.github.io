The Great Awakening

A focused archive for reading, searching, and connecting public-interest research dossiers.

The Great Awakening is a research website for visitors who want to explore complex dossiers without getting lost in scattered notes, tabs, and disconnected pages. It brings documents, summaries, categories, timelines, and related entities into one calm reading environment, so a visitor can quickly understand what a topic is about and follow the trail deeper when needed.

What The Site Helps You Do

The site is meant for structured investigation. Instead of presenting a single article and leaving the reader to piece everything together manually, it organizes material into dossiers with clear labels, searchable summaries, and contextual navigation. That makes it useful for comparing topics, revisiting sources, and building a clearer picture over time.

It is especially useful when a subject has many moving parts: institutions, events, claims, names, and background documents. The interface keeps the emphasis on reading and discovery, with quick search and an assistant-style helper available when a visitor needs a faster route through the archive.

How It Is Built

The technical side exists to keep the archive fast, searchable, and easy to host. The site uses a lightweight browser-based structure, so visitors can move between dossiers without waiting on a heavy application shell.

Hash-Based SPA Routing

A custom router (js/router.js) tracks hash modifications (e.g., #home, #dossiers, #dossier/slug). It dynamically loads fragment partials from pages/fragments/ and injects dossier HTML from pages/dossiers/ into the content shell, extracting styles to preserve unique layouts.

JSON Database Layer

A centralized flat-file database (data/dossiers.js) catalogs metadata for all research records. The Dossier Manager (js/dossier-manager.js) reads this file to build index lists, update the dynamic mega menu, and drive the local search engine.

AI Chat & Theme Utilities

An inline chat module (js/ai-chat.js) powers the helper drawer widget. User options and theme scales are governed by the theme-manager.js, saving selections to localStorage for seamless persistence.

Technical Stack

  • Custom Hash SPA Router
  • Flat-File JSON Database Layer
  • AI Assistant Chat Drawer
  • HTML5 Canvas Particle Simulation
  • Theme Manager (Dark & Light)
  • Vanilla ES6 JavaScript & CSS3

Getting Started

Clone the source and serve the directory using a lightweight web server (e.g. python -m http.server 8000 or VS Code Live Server) to enable fetch() functionality for the dynamic dossier fragments.
