IVIDS Music

An ad-free music player for searching, discovering, and listening across Android, PC, and web.

IVIDS Music is built for listeners who want a cleaner way to find and play music without the usual clutter of ads, forced subscriptions, and overloaded streaming interfaces. It combines search, playback, favorites, history, album views, and responsive layouts into one player that can run on Android, PC, TV-style screens, or the web.

Sleek, Ad-Free Listening

IVIDS Music was created to make music discovery feel direct again. Search for artists, albums, and tracks, start listening quickly, keep a personal list of favorites, and return to recent plays without moving through a maze of promotional panels.

The app is useful on small and large screens alike. On mobile it behaves like a touch-first player, on TV it supports remote-friendly movement and screensaver behavior, and on desktop it gives you a focused listening dashboard with adjustable interface scale.

Ad-Free Streams

Streams audio directly from YouTube via privacy-friendly Invidious instances with zero commercial interruptions.

UI Scaling & Color Tinting

Segmented UI scaling from 75% to 150% and Canvas-based real-time UI tinting matched to album cover art.

TV D-Pad & Screensaver

Custom spatial navigation engine for D-pad control and automated screensaver transition after 60s of inactivity.

How It Works

Behind the listening experience, IVIDS Music uses a hybrid setup that lets the same player run across several platforms while still keeping platform-specific features available.

Smart Environment Interception

On Android, a Kotlin backend intercepts WebView requests to bypass CORS limitations and resolve YouTube audio queries natively. On the standard web, the JS core automatically switches to public CORS proxies to maintain full search and discovery functionality.

Interface scale management

The UI uses CSS custom properties registered via @property to apply scale adjustments globally. Users can toggle scaling in the Settings menu (75% to 150%), which persists in localStorage and dynamically adjusts the layout density.

Personal Library Dashboard

Includes a "You" dashboard which manages user favorites, liked albums, and playback history (capped at 20 tracks, sanitized against duplicates). An offline queue system allows immediate sequential or shuffled playback.
