AI-Powered Library Restoration

Maintaining a massive local music library is a labor of love that often turns into a logistical nightmare. Between fragmented filenames, missing metadata tags, and inconsistent folder structures, libraries can quickly become unmanageable. Music File Organiser was developed to automate this restorative process using the power of modern artificial intelligence.

By combining robust file system operations with the predictive capabilities of Google's Gemini AI, we've created a tool that doesn't just rename files—it understands them. Whether your library consists of decades of legacy MP3s or high-fidelity FLAC archives, our system ensures that every track is perfectly tagged and filed in its rightful place.

How It Works

Music File Organiser scans the folder you choose, reads whatever metadata already exists, and asks Gemini to help infer missing artist, title, album, year, and track information from filenames and surrounding context. You can then use those suggestions to create a cleaner library structure without manually fixing every file one by one.

The Power of Gemini in Metadata Prediction

Traditional taggers often fail when filenames are incomplete, inconsistent, or full of personal naming conventions. Gemini helps interpret those messy names and produce usable metadata suggestions, making the app handy for old downloads, ripped CDs, archive folders, and mixed-format collections.

File System Automation & Safety

Handling thousands of personal files requires extreme caution. Our system uses a "copy-and-verify" workflow rather than "move-in-place," ensuring that original files are never compromised until the new structure is fully validated. The Python backend integrates with native OS file systems to handle special characters and long pathnames, ensuring cross-platform compatibility.

Future Roadmap

The next iteration of Music File Organiser will introduce an "Album Art Harvester," which will automatically search for and embed high-resolution cover art based on Gemini's predicted tags. We are also working on a "Duplicate Detection" algorithm that uses acoustic fingerprinting (chromaprint) to identify identical tracks regardless of their filename or current metadata quality.
